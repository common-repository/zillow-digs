This widget allows you to display your boards from Zillow Digs on your website.

== Description ==

On [Zillow Digs](http://zillow.com/digs), find incredible home design ideas and photos. Explore thousands of the very best solutions for interior design and home improvement and save your favorite design ideas on your personal Digs boards. Then use the Zillow Digs widget for wordpress to showcase your Digs board on your website in a gallery format.

*	Widget display is fully responsive and will adapt to the width of your theme's container column
*   Display as many different Digs boards as you want, each in a separate widget instance
*	Customize the maximum number of images to show in each board

== Installation ==

1. Install the Zillow Digs widget one of three ways: via the Wordpress.org plugin directory, by clicking the "Add New" button in your wordpress admin Plugins section, or by uploading the plugin files to your server in the "wp-content/plugins" directory.
2. Activate the Zillow Digs widget in the Plugins section of wordpress admin.
3. Go to Appearance -> Widgets to add a new Zillow Digs widget.
4. Enter the email address associated with your Zillow account and click the link "Submit email and load board options". The boards that you have previously created on Zillow Digs will then appear in the select dropdwon.
